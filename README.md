# 💍 Convite de Casamento

Projeto de convite digital de casamento desenvolvido com **HTML**, **CSS** e **JavaScript**, com foco em design responsivo e experiência do usuário.

## ✨ Funcionalidades

* ⏳ Contagem regressiva para o evento
* 📱 Confirmação de presença via WhatsApp
* 🎁 Lista de presentes com links externos
* 💳 Área para contribuição via Pix
* 📍 Informações sobre local e horário
* 📱 Interface responsiva para desktop e dispositivos móveis

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 🎯 Objetivo

Criar uma landing page elegante e funcional para um convite digital de casamento, utilizando apenas tecnologias web nativas, com compatibilidade para diferentes tamanhos de tela.
